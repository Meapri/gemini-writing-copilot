# v0.2.0

Major writing-agent upgrade.

## Features

- Add `--task auto` with deterministic task inference and automatic style
  profile defaults.
- Add Gemini-authored task templates and product positioning.
- Add safe project-context collection for PR descriptions, release notes,
  README/docs prose, proposals, and technical documentation.
- Add `--quality-gate` source-grounding checks for placeholders,
  model meta-commentary, and unsupported numbers or dates.
- Add `scripts/evaluate_routing.py` with local routing acceptance cases.
- Improve user-facing repair hints for Antigravity, login, timeout,
  rate-limit, empty-response, and quality-gate failures.

## Validation

- Unit test suite covers routing, project context, quality gate, templates, and
  failure UX.
- Routing eval covers common Korean and English prose-trigger requests.
