# v0.1.0

Initial public release of Gemini Writing Copilot.

## Features

- Route explicit writing tasks through Google Antigravity `agy`.
- Use Gemini 3.1 Pro High as the recommended Antigravity writing model.
- Add writing-specific tasks for emails, announcements, blogs, PR descriptions,
  release notes, READMEs, proposals, product copy, and technical docs.
- Add Gemini-generated writing guidance and built-in style profiles.
- Add output modes, voice preservation, structure, rewrite strength, and variant
  controls.
- Keep Gemini Web cookie support as an explicit fallback.

## Validation

- Plugin validation passes.
- Unit test suite passes.
- Antigravity smoke test passes.
